# **App Name**: StayCentral

## Core Features:

- Hotel Registration: Allows hotels to register with a unique email ID, admin name, password, hotel name, and automatically generated Hotel ID. Stores location for future features.
- Dashboard Overview: Displays key metrics like Total Rooms, Occupied, Waiting for Check-in, Cleaning Required, and Out of Order.
- Room Management: Enables admins to manage individual rooms, including check-in, stay details, and setting room status (e.g., Waiting for guest).
- Digital Service Order Platform: Guests can scan a QR code in their room, enter their Stay ID, and access a digital platform to order services and view bills.
- Stay ID Generation: Generates a unique Stay ID upon guest check-in, shared via SMS/WA, to track all services and bills associated with their stay.
- Dynamic File Configuration: The application generates files related to a Hotel ID: Rooms, F&B, Maintenance, Timings, New Service Creation, Team configuration, Guest Stay & Details, billing functions and corporate clients, through generative ai tool which gives suggestion to incorporate those items in those file configuration and generates the files.
- Billing and Payment: Supports corporate billing, emailing bills, marking bills as paid, editing/adding discounts, and partial payments for long stays.
- Centralized Admin Dashboard: Real-Time Room Grid, Live Service Queue, Emergency Alert System, Comprehensive Room Management.
- Dedicated Staff Dashboards: Department-Specific Queues, Manual Order Entry.
- Advanced Room & Menu Configuration: Bulk Room Creation, AI Menu Scanner, AI Experience Finder, Service Timings Management.
- Staff & Billing Management: Team Management, Corporate Client Management, Centralized Tax & Billing Configuration.
- Analytics and Reporting: Revenue Analytics, Itemized Sales Data, Complete Checkout History.
- Instant QR Code Login: Guests can scan a QR code in their room to instantly access the guest portal without needing to download an app or remember a password.
- Unified Service Hub: A beautiful, icon-driven home screen that provides one-tap access to all hotel services.
- In-Room Food Ordering: Guests can browse a full digital F&B menu and place orders directly from their phone.
- On-Demand Service Requests: Amenities, Laundry, Maintenance requests.
- Real-Time Bill & Stay Tracking: Guests have full transparency into their stay with live, itemized lists of charges.
- AI-Powered Local Exploration: The Explore feature uses AI to provide curated recommendations for local points of interest.
- Emergency SOS Button: A prominent, always-visible SOS button allows guests to instantly send a critical alert to the hotel staff with their room number.

## Style Guidelines:

- Primary Color: A strong, vibrant blue/purple (hsl(243 79% 59%)), used for main actions and highlights in both light and dark modes.
- Background Color: White (hsl(0 0% 100%)) in light mode, and a soothing dark blue (hsl(224 71% 4%)) in dark mode for the main page background.
- Foreground Color: A dark, almost black color (hsl(222.2 84% 4.9%)) in light mode, and a light, off-white color (hsl(210 40% 98%)) in dark mode for the primary text.
- Accent Color: A light gray (hsl(210 40% 96.1%)) in light mode and a deep blue (hsl(224 71% 14%)) in dark mode, used for hover states and subtle highlights.
- Use the default fonts provided by ShadCN UI which prioritize readability and accessibility.
- Simple, clear icons from a library like Lucide or Feather to represent room status and services, consistent with ShadCN UI.
- Clean and intuitive layout utilizing ShadCN UI's grid and spacing utilities for a clear visual hierarchy, making navigation easy for hotel staff.
- Subtle transitions and animations provided by Radix UI, to provide feedback on user interactions, enhancing the user experience.
